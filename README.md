# AI Voice & Writing Assistant

An AI-powered web application that enhances written content using AI. Users can input text and apply transformations such as summarization, grammar correction, formal tone conversion, and text expansion.

---

## ✨ Features

- 📝 Text Summarization
- ✅ Grammar Correction
- 💼 Formal Tone Conversion
- 📖 Text Expansion
- 📋 Copy Output
- 🗑️ Clear Input
- 📜 History Tracking
- 🎤 Voice Input (Prototype)

## 🚀 Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
